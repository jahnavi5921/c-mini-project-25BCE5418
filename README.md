## Project Description
This project is a Bus Ticket Reservation and Cancellation System developed in C programming language.

The program is menu-driven and allows users to:
- Book bus tickets
- Cancel tickets
- Search ticket details
- Display available seats
- Generate daily ticket reports

The system uses:
- Structures
- File handling
- Multiple user-defined functions
- Input validation

## Files in Repository
- main.c – C program source code
- index.html – WebAssembly compiled webpage
- index.js – JavaScript runtime file
- index.wasm – WebAssembly binary file
